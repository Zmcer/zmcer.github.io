养机场日记 v2board搭建

发表于2021-02-04|更新于2021-05-09|技术
|阅读量:

V2Board（Way to board）是一种支持多种协议的面板，我选择了它是因为快速和低占用的特点Github地址
Let’s go!
本文环境:Centos7

安装aaPanel

请在 aaPanel选择你的系统安装方式。这里以Centos为例

安装完成后我们登陆 aaPanel 进行环境的安装
选择使用LNMP的环境
Nginx 1.17
MySQL 5.6
PHP 7.3
选择 Fast 快速编译后进行安装。

安装Redis

aaPanel 面板 > App Store > 找到PHP 7.3点击Setting > Install extentions > redis 进行安装。

解除被禁止的函数

aaPanel 面板 > App Store > 找到PHP 7.3点击Setting > Disabled functions 将 putenv proc_open pcntl_alarm pcntl_signal 从列表中删除。

添加站点

aaPanel 面板 > Website > Add site

安装V2Board

通过SSH登录到服务器后访问站点路径如：/www/wwwroot/你的站点域名

注：以下命令都需要在站点目录进行执行。

配置站点目录及伪静态

添加完成后编辑添加的站点 > Site directory > Running directory 选择 /public 保存
添加完成后编辑添加的站点 > URL rewrite 填入伪静态信息

配置定时任务

aaPanel 面板 > Cron

常见问题

Q：500错误
A：检查站点根目录权限，递归755，保证目录有可写文件的权限，也有可能是Redis扩展没有安装或者Redis没有按照造成的。你可以通过查看storage/logs下的日志来排查错误或者开启debug模式。
